# Modern UI/UX Design Skill for AI Assistants

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A comprehensive, production-ready UI/UX design system skill for building premium administrative dashboards. This skill provides a complete set of patterns, components, and design tokens to generate high-quality, RTL‑first, glassmorphic interfaces.

## ✨ Features

- **RTL‑first Layout** – Optimized for Arabic, Kurdish, and other right‑to‑left languages.
- **Glassmorphic Design** – Frosted glass cards with backdrop blur and soft shadows.
- **Bento Grid System** – Modular, masonry‑style responsive layouts with staggered animations.
- **Dark Mode** – Seamless theme switching with View Transitions and shockwave effects.
- **Dual‑Currency Support** – Built‑in IQD/USD handling with live conversion.
- **Advanced Components** – Multi‑step wizards, inline editors, date range pickers, and notification systems.
- **Keyboard Accessibility** – Full keyboard support with visual shortcut hints.
- **Responsive** – Works flawlessly on mobile, tablet, and desktop.

## 🧠 How to Use This Skill

This skill is designed to be used as a prompt for AI assistants (like ChatGPT, Claude, or Copilot) to generate high‑quality UI code.

1. Copy the content of [`SKILL.md`](SKILL.md) into your AI conversation.
2. Request a specific UI component or full dashboard.
3. The AI will output React + Tailwind CSS code following the design system.

You can also use the skill as a reference for your own front‑end projects.
